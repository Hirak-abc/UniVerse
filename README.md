## 📌 Problem Statement

Current digital platforms within universities are **fragmented**, **impersonal**, and lack **real-time personalization** or **student engagement**. Students often miss out on:

- Campus events  
- Academic opportunities  
- Collaborations  
- Important announcements  

There is also **no structured system** to reward participation or track a student's **skill development** in academics, coding, or extracurriculars.

---

## 🎯 Objective

To build an **AI-powered**, university-exclusive **social and academic platform** that:

- Centralizes discussions, placement updates, and resources  
- Tracks skill-based performance  
- Encourages collaboration and learning through gamification  
- Builds a strong sense of student community  

---

## 💡 Proposed Solution

**UniVerse** is a secure **web/mobile platform** accessible only through **university email credentials**. It merges **social interaction with academic functionality**, integrating **AI-based personalization** and **gamification**.

### Highlights:

- Event updates and announcements
- Skill progress tracking via GitHub/LeetCode
- Real-time collaboration and matchmaking
- Points, badges, and leaderboards to drive engagement

---

## ✅ Expected Outcomes

- ✅ Streamlined and centralized communication  
- ✅ Visible merit-based recognition  
- ✅ Enhanced campus involvement and student life  
- ✅ Increased learning motivation through personalized AI and gamified tasks  

---

## 🚀 Key Features

| Feature | Description |
|--------|-------------|
| 🎯 Personalized AI Feed | Sentiment-aware, interest-based news and updates |
| 📚 Academic Corner | Notes, resource-sharing, shared calendar |
| 🤖 AI Study Assistant | Chatbot for doubt solving, summarizer, study planner |
| 🤝 Collaboration Hub | Auto team matcher for projects, GitHub integration |
| 🧠 Mentorship & Alumni Connect | Bridge for knowledge and opportunity sharing |
| 🛍️ Campus Marketplace | Buy/sell student goods internally |
| 📅 Events & Clubs | RSVP system, badges for participation |
| 🎮 Daily Challenges & Gamification | Fun quizzes, coding, and reward-based activities |
| 🧠 Mental Health Support | Safe anonymous chat system (planned) |
| 🧭 Lost & Found | Search and reporting system |
| 🚨 Campus Alerts & Complaints | Secure internal issue reporting platform |

## 📢 Contribution & Setup

To contribute:

```bash

git clone https://github.com/Hirak-abc/UniVerse.git

# Setup virtual environment
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate (Windows)

# Install dependencies
pip install -r requirements.txt

python app.py
```
